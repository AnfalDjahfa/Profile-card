# 🪪 Professional Profile Card

A clean, responsive, and modern Profile Card built with pure HTML and CSS, featuring Tabler Icons and a personalized dark/red aesthetics. 

## 🚀 Live Demo
You can check out the live preview here: **https://anfaldjahfa.github.io/Profile-card/**

## 🎨 Features
* **Fully Responsive:** Looks great on all screen sizes (Desktop and Mobile).
* **Modern UI:** Styled using custom CSS properties with smooth button styling.
* **Interactive Elements:** Includes hover effects on social media icons and active buttons.
* **Clean Typography:** Built with modern fonts and perfectly centered layout using Flexbox.

## 🛠️ Tech Stack & Tools
* **Frontend:** HTML5, CSS3 (Flexbox)
* **Icons:** https://tabler.io/icons
* **Hosting:** GitHub Pages
